Trek CLI is a tool for managing FiveM script workspaces. It provides a
local HTTP dashboard for browsing and editing script configurations
and themes, a CLI for workspace operations, and authentication with
the trek service.

Built in Rust and distributed as a single binary with an embedded React
frontend.

- **[Installation](/docs/cli/install)** — install via npm, shell script, or from source
- **[CLI Reference](/docs/cli/cli)** — commands, configuration, and API endpoints
