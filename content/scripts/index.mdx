Trek Script is a collection of FiveM (GTA V multiplayer mod) script libraries
and frameworks for building server resources. These libraries provide
reusable, well-tested modules for common game server patterns —
configuration management, and more.

Built in Lua and integrated into your existing FiveM server setup.

- **[Trek Core](/docs/scripts/trek-core)** — foundational library for script configuration, themes, and utilities
